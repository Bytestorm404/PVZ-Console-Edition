# Plants vs Zombies - Console Edition

![GitHub release (latest by date)](https://img.shields.io/github/v/release/Bytestorm404/PVZ-Console-Edition?include_prereleases&color=green&label=version)
![Java Version](https://img.shields.io/badge/java-26-orange)

A retro, text-based *Plants vs Zombies* port running entirely inside the console line. This project focuses on translating the classic grid combat and resource management mechanics into a terminal-friendly turn system.

---

## Current Features (v0.1.0-alpha)

* **Initial Roster:** Fully playable Peashooter `[P]` and Sunflower `[S]`.
* **Turn-Based Engine:** Grid mechanics with automatic horde progression and lane-based combat.
* **Zombie Spawns:** Random lane spawns activated from turn 4 onwards.

---

## 🛠️ Requirements & Installation

### Prerequisites
* **Java SDK 26** or higher is strictly required to run or compile this project.

### How to Play
1. Download the latest `PVZ.jar` from the [Releases](https://github.com/Bytestorm404/PVZ-Console-Edition/releases) section.
2. Open your terminal or command prompt in the folder where the file was downloaded.
3. Run the following command:
   ```bash
   java -jar PVZ.jar
